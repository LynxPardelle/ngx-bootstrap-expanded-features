Border

You can put borders in an element with only 3 classes, but you can use more to get more personalization, we will check the 3 types of border classes first, then we will se some examples.

Border Width

The width of the border can be determined with the following classes:

  • bef-bw-value = border-width: value
  • bef-bwt-value = border-top-width: value
  • bef-bwb-value = border-bottom-width: value
  • bef-bws-value = border-left-width: value
  • bef-bwe-value = border-right-width: value
  • bef-bwx-value = border-right-width: value ; border-left-width: value
  • bef-bwy-value = border-top-width: value ; border-bottom-width: value

Where value can be changed for any value.

Border Style

The style of the border can be determined with the following classes:

  • bef-bs-value = border-style: value
  • bef-bst-value = border-top-style: value
  • bef-bsb-value = border-bottom-style: value
  • bef-bss-value = border-left-style: value
  • bef-bse-value = border-right-style: value
  • bef-bsx-value = border-right-style: value ; border-left-style: value
  • bef-bsy-value = border-top-style: value ; border-bottom-style: value

Where value can be changed for any value.

Border Color

The color of the border can be determined with the following classes:

  • bef-bc-value = border-color: value
  • bef-bct-value = border-top-color: value
  • bef-bcb-value = border-bottom-color: value
  • bef-bcs-value = border-left-color: value
  • bef-bce-value = border-right-color: value
  • bef-bcx-value = border-right-color: value ; border-left-color: value
  • bef-bcy-value = border-top-color: value ; border-bottom-color: value

Where value can be changed for any value.

Radius

To use the border-radius you need to use the class bef-rounded- and add a valid value like in the following example:

Example

Paragraph with a border-radius of 30px.

Code

<p class="bg-dark text-light p-3 bef bef-r-30px" >
 Paragraph with a border-radius of 30px.
</p>
